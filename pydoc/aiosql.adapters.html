static process_sql(_query_name, _op_type, sql)¶

Pass through function because the sqlite3 driver already handles the :var_name “named style” syntax used by aiosql variables. Note, it will also accept “qmark style” variables.

Args:

_query_name (str): The name of the sql query. Unused. _op_type (aiosql.SQLOperationType): The type of SQL operation performed by the sql. sql (str): The sql as written before processing.

Returns:

str: Original SQL text unchanged.
